Complete Infantry Unit Sets 成套步兵部队合集 Комплектные наборы пехотных подразделений

Complete infantry unit paper model sets based on historical Soviet Red Army organization during the Second World War.

基于第二次世界大战时期苏联红军历史编制制作的成套步兵部队纸模合集。

Комплектные наборы бумажных моделей пехотных подразделений Красной Армии периода Второй мировой войны.

  • 1940 Red Army Rifle Battalion Set (Summer) 1940年苏联红军步兵营整营版（夏季） Набор стрелкового батальона Красной Армии 1940 года (летний)
  • 1940 Red Army Rifle Battalion Set (Winter) 1940年苏联红军步兵营整营版（冬季） Набор стрелкового батальона Красной Армии 1940 года (зимний)
